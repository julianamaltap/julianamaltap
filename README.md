## 🗂️ Projetos

### [Análise de Campanhas Google Ads — Shopping Mall](https://github.com/julianamaltap/marketing-ads-campaign-analysis)
Análise exploratória e de performance de 5 meses de campanhas Google Ads em um modelo de afiliado. O projeto cobre desde a exploração dos dados até a aplicação de KMeans para segmentação de campanhas por perfil de performance.

**Principais achados:**
- 80% das campanhas operavam no prejuízo estrutural
- Identificação de oportunidade perdida na Black Friday com ROI de 1650%
- Argumento quantificado para renegociação de comissão com base em $ 18,96 de venda gerada por dólar investido

**Ferramentas:** Python, SQL, Spark, Pandas, Numpy, Matplotlib, Seaborn, Scikit-learn
